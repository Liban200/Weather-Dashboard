# 06 Server-Side APIs: Weather Dashboard

## Description

A five day Weather Forecast that displays the weather at the exact moment and for the next five days. 
It also can display the weather for any city accros the globe.

## Technology Used
HTML
Openweather Api
CSS
JavaScript
